## Conjunto de Problemas 0

### Qué Hacer

1. Envía este [https://docs.google.com/forms/d/e/1FAIpQLScc9hTBtCnfeztHm2VgJ4if0D5rLm6ubYJ7VBk3Jf7V_wZoUg/viewform](formulario).
2. Visita este [https://submit.cs50.io/](URL), inicia sesión con tu cuenta de GitHub y haz clic en **Authorize CS50**.
3. Marca la casilla indicando que deseas otorgar al personal del curso acceso a tu envío y haz clic en **Join course**.
4. Inicia sesión en **cs50.dev**, que es una versión en la nube de Visual Studio Code (VS Code) que te proporciona tu propio “espacio de código” con todo lo que necesitas para el curso preinstalado. ¡No es necesario descargar e instalar VS Code o Python en tu propia Mac o PC!
5. Una vez que tu espacio de código haya cargado, cierra cualquier pestaña de bienvenida que se haya abierto por defecto.
6. Ejecuta `update50` en la ventana de terminal de tu espacio de código para asegurarte de que tu espacio de código esté actualizado; si se te solicita, haz clic en **Rebuild now**.
7. Envía todos los problemas a continuación:
   - Indoor Voice
   - Playback Speed
   - Making Faces
   - Einstein
   - Tip Calculator
